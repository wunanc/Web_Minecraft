## 环境要求

DomFly 插件运行所需的环境：

- `Java17` 或更高版本；
- `Paper/Folia(及其分支)` `1.20.1` 或更高版本；

::: warning
Domfly 插件未对 `spigot/bukkit` 经过针对性支持,不确保可以稳定运行
:::

## 安装

将下载好的插件文件放入服务器的 `plugins` 文件夹中，然后重启服务器即可。

不过本插件支持热加载~
